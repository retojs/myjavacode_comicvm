Prinzipien und Prioritäten der Bild-Komposition===============================================Das Suchen nach Default-Werten ist ein Suchen nach einem Ideal von Einfachheit.Bei der Komposition der PlotItems innerhalb eines Panels stellt sich die Frage, nach welchen Prioritäten die "richtigen" (default) Positionen von Figuren, Objekten und Hintergrund bestimmt wird.ComicVM berechnet die Default-Positionen und -Grössen bzw. -Distanzen wie folgt:===========07.09.2014:-----------Nachdem für eine Szene die PlotItems auf die Panels verteilt sind und damit die Reihenfolge der Figuren "in order of appearence" festgelegt ist, werden Figuren und Hintergrund wie folgt positioniert:1. Figuren: 	Sie sollen auch im kleinsten Panel gut sichtbar sein.	Also wird ihre Position so festgelegt, dass die bounding box aller im kleinsten Panel sichtbaren Figuren gerade ins Panel passt.	Für alle anderen Panels mit demselben Hintergrund wird die Grösse der Figuren beibehalten.2. Hintergründe:	Sie sollten jedes Panel voll ausfüllen.	Also wird ihre Position so festgelegt, dass sie das grösste Panel gerade ausfüllen.	Da die Figuren ihre Grösse für einen bestimmten Hintergrund beibehalten, wird auch die Grösse der Hintergründe beibehalten.===========31.7.2014:---------1.) Position der Charaktere (Motivation: alle sollen gut sichtbar sein.)2.a) Position und Grösse des Hintergrunds (Motivation: konsistent mit vorangehenden Panels)2.b) Grösse der Charaktere Wobei die Grösse der Charaktere natürlich die optimale Position beeinflusst...--> Idee: 1.) Proportionen der Bounding Box der Charaktere bestimmen (= Distanz plus Grösse, muss konsistenz sein mir vorangehenden Panels)	Bounding Box optimal* im Panel plazieren (siehe unten)2.) Position und Grösse des Hintergrunds entsprechend bestimmen. => Prinzip 1: "The Panel contains the bounding box of all its characters." => Prinzip 2: "The characters' positions are consistent in each background."Die Reihenfolge der Figuren bleibt natürlich auch erhalten. (Order of Appearence)Somit müssen ev. Figuren gezeichnet werden, die sich zwischen den Akteuren in diesem Panel befinden. => Printzip 3: "Characters appear in order of appearence." *) Bounding Box *optimal* im Panel plazieren heisst:	- sie soll die selbe absolute Grösse behalten (Analogie zu Film: Kamera bleibt per Default still)		- falls die Hintergrund-Grösse dazu nicht ausreicht: 		- skalieren, so dass der Hintergrund das Panel gerade ausfüllt (mit Rand)		 => Prinzip 4: "Camera maintains character size if possible."Zum Bestimmen der idealen (default) proportion zwischen Character und Hintergrund, kann ein Master-Panel definiert werden.Hier werden Charactere und Hintergrund ideal plaziert. Alle anderen Panels richten dich nach der daraus resultierenden Proportion zwischen Charaktere und Hintergrund. => Prinzip 5: "The Master-Panel defines the right proportion of characters and background."  Zusammenfassung:---------------- => Prinzip 1: "The Panel contains the bounding box of all its characters." => Prinzip 2: "The characters' positions are consistent in each background." => Prinzip 3: "Characters appear in order of appearence." => Prinzip 4: "Camera maintains character size if possible." => Prinzip 5: "The Master-Panel defines the right proportion of characters and background." --- older:1. Bei mehreren Figuren oder Objekten muss ihre Distanz ideal sein (default). 2. Daraus ergibt sich die Grösse und Position des Hintergrunds entsprechend dem vorangegangenen Panel.   sowie die Grösse der Charaktere selbst.   Bei einer einzigen Figur wird ihre Grösse per default gesetzt (ideal, bzw. style-dependent).3. die Anordnung der Figuren (Reihenfolge) entspricht 	a) dem vorangegangenen panel	b) falls nicht vorhanden: der reihenfolge ihres auftretens.---defaults: {		figuren: {				bounding-box: goldener schnitt der bildbreite gleichmässig ausfüllend		vertical-pos: 50%		single-size: inverser goldener schnitt	}				bgr: verzerrungsfrei auf panelgrösse gezoomt}
